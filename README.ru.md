# Кира — локальный голосовой ассистент для macOS

Голосовой (или текстовый) ввод → локальная LLM с tool calling → действия на маке → голосовой ответ.
Всё работает офлайн (кроме погоды и веб-поиска), ничего не уходит в облако.

## Стек

| Компонент | Что используется |
|---|---|
| LLM | Qwen3-1.7B (Q4, ~1.4 ГБ) через Ollama — ~44 ток/с на M4, влезает даже в 8 ГБ M1. `--model` меняет модель, `--think` включает размышления |
| Поиск в интернете | DuckDuckGo (без ключей) или Ollama Web Search API при наличии `OLLAMA_API_KEY`; сайты-источники показываются в панели |
| Распознавание речи | Whisper-small на MLX (~500 МБ, все популярные языки) |
| Озвучка | нативный `say` (голос Milena для русского) |
| Действия на маке | tool calling → AppleScript (`osascript`), `open`, `pmset`, Shortcuts CLI |

## Запуск

```bash
# один раз: установить Ollama и модель (если ещё нет)
brew install ollama
ollama pull qwen3.5:2b-q4_K_M

# запуск
uv run python -m kira --ui         # интерфейс со сферой + вызов по имени «Кира»
uv run python -m kira              # текстовый ввод, голосовой ответ
uv run python -m kira --voice      # голосовой диалог без имени
uv run python -m kira --wake       # вызов по имени, но в терминале
uv run python -m kira --no-speak   # тихий текстовый режим
```

При первом запуске `--voice` скачается модель Whisper (~500 МБ) и macOS
спросит разрешение на доступ к микрофону для терминала.

## Что умеет

Спросите своими словами, на любом языке:

**Информация**: «который час в Токио», «какая погода в Питере», «что нового
в мире» (RSS-дайджест), «сколько стоит биткоин» (веб-поиск с источниками),
«сколько будет 15% от 2400» (точный калькулятор), «переведи 100 долларов
в рубли» (живой курс), «сколько батареи», «сколько места на диске»

**Действия на маке**: «открой Safari», «закрой калькулятор», «поставь
громкость на 30», «следующий трек» (Spotify/Music), «заблокируй экран»,
«что у меня в буфере обмена», «скопируй …», «найди файл отчёт» (Spotlight),
«запусти быструю команду <имя>»

**Продуктивность**: «поставь таймер на 10 минут на пиццу» (голосовое
оповещение), «напомни завтра в 10 купить хлеб» (Напоминания), «добавь
встречу в календарь на пятницу в 15:00», «что у меня сегодня в календаре»,
«создай заметку …», «отправь сообщение маме …» (iMessage)

**Разговор**: «расскажи сказку про кота» (сочиняет сама, без поиска),
переводы, любые вопросы

Частые запросы (время, арифметика, буфер, цены, новости) обрабатываются
детерминированными маршрутами мимо LLM — мгновенно и без галлюцинаций.

Новые действия добавляются в `kira/tools.py`: обычная python-функция с
докстрингом + строчка в списке `TOOLS`.

## Как устроен голосовой режим

`kira/listen.py` слушает микрофон блоками по 30 мс, полсекунды калибруется
под фоновый шум, ждёт начала речи и останавливает запись после 1.2 с тишины.
Фраза уходит в Whisper, текст — в LLM. Пока Кира говорит, микрофон не
слушает (иначе она услышала бы саму себя).

## Вызов по имени

Без сторонних сервисов: детектор речи почти не ест CPU (просто громкость),
а услышав фразу, Whisper распознаёт её локально и ищет обращение «Кира»
(в любом падеже, можно и «Kira»). Работает так:

- «Кира, открой сафари» — сразу выполняет
- «Кира!» — ответит «Да?» и выслушает команду
- разговоры без имени полностью игнорируются (и никуда не отправляются —
  всё распознаётся локально)

## Интерфейс (`--ui`)

Плавающая полупрозрачная панель поверх всех окон (PySide6): светящаяся
сфера дышит в ожидании, пульсирует от громкости голоса, крутит дугу, пока
модель думает, и меняет цвет по состоянию. Панель появляется с плавным
фейдом при слове «Кира» и тает через несколько секунд после ответа.
Esc — спрятать панель, правый клик — выйти, Ctrl+C в терминале — тоже выход.
